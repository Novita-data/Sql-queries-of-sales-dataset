#  A beginner Sql-queries-of-sales-dataset

## Project Overview
This project explores and analyzes a Sales Superstore dataset using SQL in SQLite.  
The goal of the project is to practice SQL querying, data analysis, and business-oriented problem solving using sales data.

The project focuses on:
- Sales performance analysis
- Customer analysis
- Product analysis
- Category analysis
- Time-based sales analysis
- Query writing and debugging practice

---

# Tools Used

- SQLite
- DB Browser for SQLite
- SQL

---

# Skills Demonstrated

This project includes practice with:

- SELECT statements
- WHERE filtering
- GROUP BY
- ORDER BY
- LIMIT
- Aggregate functions
  - SUM()
  - AVG()
  - COUNT()
- Aliases using AS
- Date extraction using substr()
- Time-based analysis
- SQL debugging and syntax correction
